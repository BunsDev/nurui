### Props

| Prop Name      | Type     | Default                                                  | Description                                                             |
| -------------- | -------- | -------------------------------------------------------- | ----------------------------------------------------------------------- |
| `trailColor`   | `string` | `"#D0FBB6"`                                              | Color of the trail (supports HEX format).                               |
| `dotSize`      | `number` | `4`                                                      | Size of each square dot in pixels.                                      |
| `fadeDuration` | `number` | `600`                                                    | Time in milliseconds for trail fade effect. Lower means faster fading.  |
| `className`    | `string` | `"w-full h-full fixed inset-0 pointer-events-none z-50"` | CSS classes applied to the canvas element for positioning and layering. |
