### Props

| Prop            | Type                          | Default | Description                                                                |
| --------------- | ----------------------------- | ------- | -------------------------------------------------------------------------- |
| `effectOnHover` | `boolean`                     | `true`  | Enables the text  animation when the button is hovered.                |
| `effectOnLoad`  | `boolean`                     | `true`  | Triggers the text  animation once on component load.                   |
| `className`     | `string`                      | `-`     | Custom CSS classes to apply to the button.                                 |
| `children`      | `React.ReactNode`             | `-`     | Text or elements to display inside the button. Animation is applied to it. |
| `onClick`       | `(event: MouseEvent) => void` | `-`     | Function called when the button is clicked. Also triggers the animation.   |
| `...props`      | `ComponentProps<"button">`    | `-`     | Inherits all native HTML button attributes like `type`, `disabled`, etc.   |
