### Props

`FeatureCard`

| Prop        | Type                                                                                | Default | Description                                         |
| ----------- | ----------------------------------------------------------------------------------- | ------- | --------------------------------------------------- |
| `feature`   | `{ title: string; icon: React.ComponentType<SVGSVGElement>; description: string; }` | –       | Object containing the icon, title, and description. |
| `className` | `string`                                                                            | –       | Additional Tailwind CSS classes.                    |
| `...props`  | `React.HTMLAttributes<HTMLDivElement>`                                              | –       | Any native `div` props like `id`, `onClick`, etc.   |

`GridPattern`

| Prop       | Type                            | Default | Description                                                     |
| ---------- | ------------------------------- | ------- | --------------------------------------------------------------- |
| `width`    | `number`                        | –       | Width of the pattern unit.                                      |
| `height`   | `number`                        | –       | Height of the pattern unit.                                     |
| `x`        | `string`                        | –       | X offset position of the pattern.                               |
| `y`        | `string`                        | –       | Y offset position of the pattern.                               |
| `squares`  | `number[][]`                    | –       | Array of square \[x, y] positions to highlight within the grid. |
| `...props` | `React.SVGProps<SVGSVGElement>` | –       | All standard SVG element props.                                 |

`AnimatedContainer`

| Prop        | Type              | Default | Description                                   |
| ----------- | ----------------- | ------- | --------------------------------------------- |
| `children`  | `React.ReactNode` | –       | Child elements to be animated.                |
| `className` | `string`          | –       | Optional Tailwind classes for styling.        |
| `delay`     | `number`          | `0.1`   | Delay in seconds before the animation starts. |
