### Props

| Prop     | Type       | Default                       | Description                                         |
| -------- | ---------- | ----------------------------- | --------------------------------------------------- |
| `bars`   | `number`   | `20`                          | Number of vertical bars rendered in the background. |
| `colors` | `string[]` | `['#3ca2faD9','transparent']` | Gradient colors for each bar (top to bottom).       |
