### Props

| Prop        | Type              | Default     | Description                                                                |
| ----------- | ----------------- | ----------- | -------------------------------------------------------------------------- |
| `children`  | `React.ReactNode` | –           | The content to display on top of the animated background.                  |
| `className` | `string`          | –           | Additional CSS classes for the outer container.                            |
| `speed`     | `number`          | `1`         | Controls how fast the warp lines animate. Lower values = faster animation. |
| `color`     | `string`          | `"#3ca2fa"` | The color of the animated warp lines (default: blue).                      |
| `height`    | `number`          | `1`         | The thickness (height) of each warp line in pixels.                        |
| `line`      | `number`          | `50`        | Number of warp lines to render radially.                                   |
