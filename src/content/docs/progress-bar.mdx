### Props

`ProgressBarDemo`

| Prop        | Type     | Default | Description                                                   |
| ----------- | -------- | ------- | ------------------------------------------------------------- |
| `className` | `string` | `""`    | Additional CSS classes for customizing the wrapper container. |

`ProgressBar`

| Prop                  | Type     | Default    | Description                                                          |
| --------------------- | -------- | ---------- | -------------------------------------------------------------------- |
| `max`                 | `number` | `100`      | The maximum value the progress bar can reach.                        |
| `min`                 | `number` | `0`        | The minimum value the progress bar can have.                         |
| `value`               | `number` | `0`        | The current value of the progress bar.                               |
| `gaugePrimaryColor`   | `string` | `Required` | The color of the active (progress) portion of the circular gauge.    |
| `gaugeSecondaryColor` | `string` | `Required` | The color of the inactive (remaining) portion of the circular gauge. |
| `className`           | `string` | `""`       | Custom CSS classes for styling the circular progress container.      |
