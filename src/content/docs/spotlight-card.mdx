### Props

| Prop         | Type                                                 | Default     | Description                                                                |
| ------------ | ---------------------------------------------------- | ----------- | -------------------------------------------------------------------------- |
| `children`   | `ReactNode`                                          | Required    | Content to be displayed inside the card.                                   |
| `className`  | `string`                                             | `''`        | Additional custom Tailwind CSS or utility classes for the card.            |
| `glowColor`  | `'blue' \| 'purple' \| 'green' \| 'red' \| 'orange'` | `'blue'`    | Glow hue color used in the dynamic gradient.                               |
| `size`       | `'sm' \| 'md' \| 'lg'`                               | `'md'`      | Predefined size of the card (ignored if `customSize` is `true`).           |
| `width`      | `string \| number`                                   | `undefined` | Custom width (used when `customSize` is `true`).                           |
| `height`     | `string \| number`                                   | `undefined` | Custom height (used when `customSize` is `true`).                          |
| `customSize` | `boolean`                                            | `false`     | When `true`, disables preset sizing (`size`) and allows full size control. |
