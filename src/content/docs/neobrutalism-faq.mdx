### Props

| Prop       | Type        | Default                        | Description                                                                  |
| ---------- | ----------- | ------------------------------ | ---------------------------------------------------------------------------- |
| `data`     | `FaqItem[]` | `defaultFaqData`               | Array of FAQ items to display. Each item has `id`, `question`, and `answer`. |
| `title`    | `string`    | `"FAQ"`                        | The main heading text displayed at the top of the FAQ section.               |
| `subtitle` | `string`    | `"FREQUENTLY ASKED QUESTIONS"` | The subheading text displayed below the title.                               |
