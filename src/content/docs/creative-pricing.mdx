### Props

`CreativePricing`

| Prop          | Type            | Default                                    | Description                                                                   |
| ------------- | --------------- | ------------------------------------------ | ----------------------------------------------------------------------------- |
| `tag`         | `string`        | `"Simple Pricing"`                         | Small decorative text shown above the main title.                             |
| `title`       | `string`        | `"Make Short Videos That Pop"`             | Main heading text for the pricing section.                                    |
| `description` | `string`        | `"Edit, enhance, and go viral in minutes"` | Subtext shown under the title.                                                |
| `tiers`       | `PricingTier[]` | `Required`                                 | Array of pricing tiers to display. Each tier includes name, icon, price, etc. |

`PricingTier`

| Property      | Type              | Required | Description                                                       |
| ------------- | ----------------- | -------- | ----------------------------------------------------------------- |
| `name`        | `string`          | Yes      | Name of the pricing tier.                                         |
| `icon`        | `React.ReactNode` | Yes      | Icon shown at the top of the card.                                |
| `price`       | `number`          | Yes      | Monthly price for the tier.                                       |
| `description` | `string`          | Yes      | Short description of the pricing tier.                            |
| `features`    | `string[]`        | Yes      | List of features included in the pricing tier.                    |
| `popular`     | `boolean`         | No       | Marks the tier as popular; adds a badge and special button style. |
| `color`       | `string`          | Yes      | Tailwind color prefix (e.g., `"blue"`, `"amber"`, `"green"`).     |

`Button`

| Prop        | Type                                                                          | Default     | Description                                                                  |
| ----------- | ----------------------------------------------------------------------------- | ----------- | ---------------------------------------------------------------------------- |
| `variant`   | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Visual style of the button.                                                  |
| `size`      | `"default" \| "sm" \| "lg" \| "icon"`                                         | `"default"` | Controls button height, padding, and border radius.                          |
| `asChild`   | `boolean`                                                                     | `false`     | If `true`, renders using `<Slot />` for passing custom elements as children. |
| `className` | `string`                                                                      | `-`         | Additional Tailwind classes to override or extend default styles.            |
| `...props`  | `React.ButtonHTMLAttributes<HTMLButtonElement>`                               | `-`         | All standard HTML `<button>` attributes like `onClick`, `type`, `disabled`.  |
