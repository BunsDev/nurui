### Props

`Banner`

| Prop            | Type                             | Default    | Description                                                                              |
| --------------- | -------------------------------- | ---------- | ---------------------------------------------------------------------------------------- |
| `id`            | `string`                         | `-`        | HTML `id` used for accessibility and to persist close status in `localStorage`.          |
| `xColor`        | `string`                         | `-`        | Custom color for the close (X) icon.                                                     |
| `variant`       | `"normal"` \| `"rainbow"`        | `"normal"` | Style variant of the banner. `"rainbow"` applies a colorful animated background.         |
| `height`        | `string`                         | `"3rem"`   | Height of the banner.                                                                    |
| `rainbowColors` | `string[]`                       | `[...]`    | Custom color stops for the rainbow background animation. Only used in `rainbow` variant. |
| `changeLayout`  | `boolean`                        | `true`     | Whether to inject layout-affecting CSS variables (`--fd-banner-height`).                 |
| `children`      | `React.ReactNode`                | `-`        | Content to be displayed inside the banner.                                               |
| `className`     | `string`                         | `-`        | Additional Tailwind or custom classes for styling.                                       |
| `...props`      | `HTMLAttributes<HTMLDivElement>` | `-`        | Additional div props passed to the banner container.                                     |
