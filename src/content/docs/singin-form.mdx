### Props

`Input`

| Prop        | Type                                          | Default | Description                                                                 |
| ----------- | --------------------------------------------- | ------- | --------------------------------------------------------------------------- |
| `type`      | `string`                                      | `-`     | Input field type (e.g., `"email"`, `"password"`, etc.).                     |
| `className` | `string`                                      | `-`     | Additional Tailwind CSS classes for styling customization.                  |
| `...props`  | `React.InputHTMLAttributes<HTMLInputElement>` | `-`     | All other default input props like `value`, `onChange`, `placeholder`, etc. |

`(Optional) LoginForm Props (to make it reusable)`

| Prop         | Type                                        | Default         | Description                                       |
| ------------ | ------------------------------------------- | --------------- | ------------------------------------------------- |
| `onSubmit`   | `(email: string, password: string) => void` | Internal Logic  | Callback when user submits the form.              |
| `title`      | `string`                                    | "Welcome Back"  | Title displayed at the top.                       |
| `subtitle`   | `string`                                    | "Sign in to..." | Subtitle message under the title.                 |
| `showGoogle` | `boolean`                                   | `true`          | Whether to show the "Sign in with Google" button. |
