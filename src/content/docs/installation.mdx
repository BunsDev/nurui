# Installation

###### Learn how to install Bytenexia components and set up your project.

<Steps>

<Step>Add Components</Step>

You can now start adding components to your project.

<Cli
  dependencies={[
    { label: "npm", command: "npx bytenexia add gradient-button" },
    { label: "pnpm", command: "pnpm dlx bytenexia add gradient-button" },
    { label: "bun", command: "bunx bytenexia add gradient-button" },
    { label: "yarn", command: "yarn dlx bytenexia add gradient-button" },
  ]}
/>

This will generate the component in: `components/bytenexia/GradientButton.tsx`

<Step>Import and use the component</Step>

The command above will add the `GradientButton` component to your project. You can then import it like this:

<CodeBlock
code={`import { GradientButton } from "@/components/bytenexia/GradientButton";

export default function Home() {
  return (
    <div>
      <GradientButton />
    </div>
  );
}`}
/>

<Step>Optional: Components list</Step>

To see the full `list` of available components you can install:

<Cli
  dependencies={[
    { label: "npm", command: "npx bytenexia list" },
    { label: "pnpm", command: "pnpm dlx bytenexia list" },
    { label: "bun", command: "bunx bytenexia list" },
    { label: "yarn", command: "yarn dlx bytenexia list" },
  ]}
/>

This will display all the components supported by Bytenexia.

</Steps>
